# E-Commerce Flask Application

This is a simple CRUD application for user management using Node.js with Express, Mongoose and MongoDB.

## Features

### Manages following data:
- First Name
- Last Name
- Date of Birth
- Address1
- Address2
- City
- Postal Code
- Country
- Phone Number
- Email
- User Notes
